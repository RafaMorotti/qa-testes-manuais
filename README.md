# Testes Funcionais Manuais

## Função de Login

Projeto de portfólio voltado para testes funcionais manuais, com foco na validação do fluxo de login em um ambiente web.

## Objetivo

Este caso de teste tem como objetivo demonstrar o raciocínio de testes, os retornos esperados, as validações aplicadas e a sequência lógica de execução dos cenários.

## Escopo dos testes

Foi criado um e-mail exclusivo para a realização dos testes de login, permitindo a execução de:
- cenários positivos de acesso e cadastro;
- cenários negativos com validações de campos e credenciais.

## Estrutura das pastas

O repositório contém:
- prints da estrutura do caso de teste criado na plataforma Qase;
- prints da execução dos testes anexados na plataforma Qase.

## Exemplos de cenários testados

- teste negativo com e-mail vazio;
- teste negativo com senha vazia;
- teste negativo com e-mail inválido;
- teste negativo com senha inválida;
- teste positivo com credenciais corretas.

## Evidências

Na pasta `prints`, estão organizados:
- os resultados dos testes negativos;
- os resultados dos testes positivos;
- os prints da estrutura do caso de teste criado.

## Ferramentas utilizadas

- **Site para os testes:** https://automationpratice.com.br/
- **Ferramenta de captura de tela:** Lightshot
- **Plataforma de documentação e execução:** Qase
